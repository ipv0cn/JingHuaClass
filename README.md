# 精华课堂（JingHuaClass）

河南信息统计职业学院精华课堂系统的 Python 客户端工具，封装了大部分 Web 端可实现的功能，并提供了一个抢答功能的实例。

## 快速开始

1. 安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 复制示例配置文件：
   ```bash
   cp config.example.py config.py
   ```

3. 编辑 `config.py`，填入您的账号凭据

4. 启动程序：
   ```bash
   python main.py
   ```

## 使用须知

- 请妥善保管您的账号信息，避免泄露
- 使用时请遵守学校相关网络与教学管理规定
- 本项目仅用于学习与技术交流目的
