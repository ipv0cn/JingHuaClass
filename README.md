# 精华课堂（JingHuaClass）

一个面向精华课堂系统的 Python 客户端工具，支持课程管理、课堂抢答、教学评价等核心功能。

## 功能特性

- 用户登录与身份认证  
- 获取课程列表  
- 参与课堂抢答  
- 提交教学评价  
- 查询个人用户信息  

## 快速开始

1. 安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 复制示例配置文件：
   ```bash
   cp config.example.py config.py
   ```

3. 编辑 `config.py`，填入您的账号凭据

4. 启动程序：
   ```bash
   python main.py
   ```

## 使用须知

- 请妥善保管您的账号信息，避免泄露  
- 使用时请遵守学校相关网络与教学管理规定  
- 本项目仅用于学习与技术交流目的
